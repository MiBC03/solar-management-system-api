# Solar Management System (API)

Este projeto representa a refatoração de um sistema web desenvolvido inicialmente em PHP
para uma arquitetura baseada em API REST, utilizando tecnologias mais modernas.

## Contexto
Durante o estágio supervisionado, surgiu a oportunidade de reimplementar o sistema
utilizando Node.js e React, com o objetivo de acompanhar as demandas do mercado e
aprofundar conhecimentos em novas tecnologias.

## Arquitetura
- Backend: API REST desenvolvida em Node.js
- Frontend: Aplicação React para consumo da API
- Separação clara de responsabilidades entre frontend e backend

## Funcionalidades
- CRUD de clientes
- Comunicação frontend-backend via API REST
- Integração com banco de dados relacional

## Tecnologias Utilizadas
- Node.js
- TypeScript
- React
- Axios
- JavaScript

## Aprendizados
O projeto reforçou a capacidade de adaptação tecnológica, aprendizado autônomo e
compreensão de arquiteturas modernas baseadas em APIs.
